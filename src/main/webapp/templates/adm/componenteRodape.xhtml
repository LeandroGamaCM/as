Manual
Do Acesso ao Sistema
01. O usuário administrador deve logar e atualizar as credenciais do usuário padrão criado pelo sistema. Além disso deve fornecer um email válido.
02. Não há limite para o cadastro de usuários administradores. O cadastro deve ser feito por outro usuário adm, na página Usuários.
Dos Alunos
02. Para que um estudante consiga se cadastrar no Sistema PAAE é necessário que ele seja pré-cadastrado pela Assistência Social.
03. Por questões de integridade das informações, o sistema não permite a exclusão de estudantes cadastrados.
04. Cada estudante recebe um status conforme as fases concluídas. Pré-Cadastrado; Com Inscrição Pendente; Com Inscrição Realizada.
05. Os estudantes podem ser listados de acordo com alguns filtros. Esses filtros permitem mostrar apenas as turmas desejadas.
Das Turmas
06. Por conta das turmas serem volatéis, a Assistência Social deve, sempre antes do período de inscrição, atualizar as turmas cadastradas.
07. Não ocorrendo isso, o formulário socioeconômico fica sem opção de turma para os estudantes selecionarem.
Das Entrevistas
08. Os estudantes com inscrição realizada são listados na tabela de entrevistas. Nessa tabela, os formulários podem ser expandidos.
09. Os estudantes que ainda não foram entrevistados aparecem numa outra tabela, onde é possível entrevistá-los.
Do Barema
10. O sistema permite alterar o peso dos critérios socioeconômicos adotados para pontuar a situação social dos estudantes.
11. Cabe a Assistência Social atualizar o valor do salário mínimo para manter o barema atual.
Dos Relatórios
12. Os estudantes com inscrição efetuada são classificados em ordem decrescente de acordo com a pontuação do barema.
13. O sistema permite visualizar as preferências de bolsas, estatísticas da renda per capita e o status de inscrição dos estudantes.
