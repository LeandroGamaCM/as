Manual
Do Acesso ao Sistema
01. O Sistema PAAE, atento a questões de segurança, só permite o cadastro de usuários que possuem o CPF na base de dados.
02. Todos os usuários devem possuir um email válido para realizar o cadastro.
03. O usuário que esquece a sua credencial de acesso deve utilizar a função Recuperar Senha, presente na tela de acesso.
04. Recuperar a senha só é possível se o email cadastrado é válido e pertence ao próprio usuário.
05. Se acontecer a perda da conta de email, o usuário deve atualizar imediatamente seus dados cadastrais.
06. O sistema não permite a exclusão de usuários. Dados não válidos inseridos poderá comprometer o acesso ao sistema.
Da Inscrição no PAAE
07. O período de inscrição abre e fecha por única e exclusiva determinação da Assistência Social.
08. Cabe ao usuário se informar das datas que marcará o início e o término do período.
09. Fora do período de inscrição, o formulário para estudo socioeconômico fica indisponível.
10. Por se tratar de um processo de concessão de auxílios, o formulário é extenso.
11. Os dados fornecidos devem possuir veracidade. Dúvidas no preenchimeto dos campos devem ser levadas a Assistência Social.
12. Preenchido o formulário, terá o usuário duas opções: Salvar os dados preenchidos; Enviar Inscrição.
13. O usuário que ainda não possui os documentos para serem anexados devem optar por Salvar Dados.
Do Anexo de Comprovantes
14. Alguns campos do formulário necessita de documentos comprobatórios da informação fornecida.
15. Quando o usuário envia a inscrição, é chamada a tela de anexar comprovantes.
16. É preciso que os documentos estejam digitalizados para serem anexados. Cada um deve possuir no máximo 1,4mb.
17. Depois de anexar o documento, é necessário clicar em Enviar.
